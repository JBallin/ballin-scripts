# Brand assets

This folder contains source and exported assets for the Ballin visual system.

## GitHub social preview

- Source: `social-preview.svg`
- Export: `social-preview.png`
- Size: `1280x640`
- Primary identity: lowercase `ballin`
- Supporting copy: `Back up your dotfiles and update your macOS development environment.`

Keep the SVG source editable and treat the PNG as the GitHub upload artifact.
The social preview establishes the wordmark-led direction from issue #196; README
hero work in issue #193 can reuse the same matte graphite background, off-white
type, and subtle green accent without introducing a larger brand kit yet.

When refreshing the export, render the SVG at exactly `1280x640` and overwrite
`social-preview.png`. Avoid thumbnail exporters that pad or crop the source
artwork.
